ステマ対策クエスト README
作成日: 2026-06-11
概要
このフォルダには、消費者庁のステルスマーケティング規制関連資料をもとに作成した社内向け資料と実務チェック用HTMLが入っています。
主に使うファイル
1. stema_quest_all_in_one.html
   社内ガイドと4ゲートチェックリストを1つにまとめたメインファイルです。
   通常はこちらを開いて使ってください。
2. stema_internal_guide.html
   学習用の社内ガイド単体版です。
   章立ては「1. まず押さえる結論」から「11. 参照した一次情報」までです。
3. stema_review_form.html
   4ゲートチェックリスト単体版です。
   判定フォームだけを使いたい場合はこちらを使います。
4. stema_review_form_template.md
   4ゲートチェックリストのMarkdown版です。
   Notion、Google Docs、社内Wikiなどに移す場合の下書きとして使えます。
5. stema_campaign_checklist.csv
   公開前チェック項目をCSV化したものです。
   Excel、Google Sheets、管理台帳などに取り込めます。
推奨運用
通常運用では stema_quest_all_in_one.html を使います。
1. HTMLをブラウザで開く
2. 「学習ガイド」で判断軸を確認する
3. 「4ゲートチェック」で案件情報を入力する
4. 相手確認、依頼前、下書き、投稿後の順にチェックする
5. 右側の自動判定を確認する
6. 「要約コピー」で承認ログやチケットに貼り付ける
7. 「HTMLを保存」で入力済み状態のHTMLを保存する
4ゲートチェックの考え方
このチェックリストは、社内ガイドの「9. 社内の運用装置」を実務用に落とし込んだものです。
1. 相手確認
   個人、法人、社員・関係者など、誰に依頼するかを確認します。
2. 依頼前チェック
   依頼書、発注書、DM、メールにPR表記義務、表示位置、削除禁止、下書き提出、証跡提出を入れたか確認します。
3. 下書き確認
   PR・広告表記が見えるか、埋もれていないか、第三者の自主的な声と誤認されないか確認します。
4. 投稿後チェック
   公開URL、公開後キャプチャ、下書き通りか、PR表記が削除・変更されていないか、1週間後確認を管理します。
自動判定について
自動判定は、社内ガイドの「8. 実務判断フロー」をもとにしています。
特に以下の場合は、依頼前または公開前に停止・修正・法務確認が必要です。
- 当社商品・サービスの表示がある
- 当社が依頼、対価提供、内容確認、編集をしている
- 広告・PRであることが明瞭ではない
注意事項
この資料は社内実務の補助資料です。
個別案件では、実態に応じて法務・コンプライアンス確認を行ってください。
医療、美容、健康、金融、ランキング、比較、レビュー施策などは誤認リスクが高いため、早めに確認してください。
